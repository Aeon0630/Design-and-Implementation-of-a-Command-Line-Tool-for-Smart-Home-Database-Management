# 介绍
随着智能家居设备的大规模普及，海量用户行为数据与设备运行记录的高效管理成为行业痛点。传统管理系统存在数据碎片化存储、SQL操作门槛高及可视化能力薄弱等问题。本研究设计并实现了一款面向智能家居场景的数据库管理命令行工具，通过构建包含用户信息、设备记录、使用日志等5张核心表的标准化关系数据库，形成完整的数据存储体系。工具集成多维度功能模块：基于语法解析与结构校验的 SQL错误检测机制，可精准识别各类查询异常并提供修改建议；结合graphviz与matplotlib的可视化组件，实现查询执行计划树状图与结果表格的交互式展示；基于大语言模型的自然语言转 SQL模块，支持多表关联查询的语义解析与语句生成；一键重置功能可快速重建表结构并初始化测试数据，满足开发测试环境的快速部署需求。该工具通过模块化设计与标准化接口，实现从数据存储、查询校验到智能交互的全流程闭环，为智能家居数据管理提供低门槛解决方案，有效降低非技术用户的操作成本，推动数据库工具向智能化、易用化方向发展。

# 主要工作
在此项目中，首先构建了智能家居系统，并设计了各个关系的关系模式，同时完成了支持在python中对数据库进行查询和相关可视化的工作，包括判断查询语句的正确性、可视化查询流程、可视化查询结果和通过自然语言进行查询。
